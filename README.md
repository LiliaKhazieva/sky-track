# SkyTrack

✈️ — Трекинг авиаполётов.
Приложение получает данные о рейсах, отображает их на карте и предоставляет детальную информацию о каждом самолёте: скорость, высоту, направление, маршрут и статус полёта.

<img src="./screenshots_for_you/map.png" alt="screen" width="500">

Проект подходит для изучения работы с авиационными API, геоданными и асинхронными запросами, а также для создания собственных дашбордов мониторинга воздушного трафика.

Взможности:

- Получение данных о самолётах.

- Фильтрация: поиск по номеру рейса, авиакомпании, типу воздушного судна или координатам.

- Визуализация: отображение траектории полёта на интерактивной карте.

- Детализация: просмотр скорости, высоты, курса, расчётного времени прибытия.

- Кэширование: временное хранение данных для снижения нагрузки на API и ускорения отображения.
